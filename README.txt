                NITC NEWS

Clone the repository in a folder
First install all the things from requirements.txt in a virtual 
Export FLASK_APP=flaskr
Export FLASK_ENV=development
Flask init-db
If the above command shows error then
Just type flask run
Copy the url http://127.0.0.1:5000/auth/register
Register in it
U will be redirected for login.login it with registered name and password
Click on ADD TO add news
Click on #COMMENT OR TAGS to add comment for the news
Click on news to see the details
Click on edit to edit title and description of news.
Click on various tags in detail page to see all news with those tags
There is a search bar present,type the title of note and click on search button.
That's all with NITC NEWS.
